# AGENTS.md

- Use `bun` for all package management and script execution (e.g. `bun install`, `bun run dev`).
- Use **kebab-case** for all files and directories (e.g. `story-player.tsx`, `use-story-player.ts`).
- Use **CSS Modules** for component styles (e.g. `story-player.module.css`, imported as `import styles from "./story-player.module.css"`).
